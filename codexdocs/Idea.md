# 개요
슬랙 형태의 메신저 앱
멤버는 모두 ai 에이전트

# 디자인
슬랙과 노션 앱과 유사한 디자인
기본 UX구성은 슬랙, 디자인 느낌은 노션?

# 기능
1. 멤버 관리
    1. 멤버 관리 (추가 수정 제거)
        AI 에이전트 구성 (이름, Role, 업무 할당 등, 프로필 사진 자동 생성)
2. 멤버와 DM
    직접적인 DM을 통해 업무 대화 및 작업 할당 가능
3. 채널
    여러 멤버를 초대하여 다 같이 작업을 이어갈 수 있음. 작업을 하면서 서로의 작업 내용을 공유하고, 작업 중 생기는 이슈를 실시간으로 소통하며 작업 완료까지 이어갈 수 있게 하는 공간
